01 — Introduction

Getting Started

ProChorder is a browser-based tool for writing, organizing and performing songs with chords and lyrics. Everything is saved locally in your browser — no account is required for the core features.

🎵
Song Library

Create and manage an unlimited number of songs using the project selector in the top left.

🎸
Chords & Lyrics

Switch between chord mode and text mode with the CHORD button in the top right.

🌙
Light / Dark

Toggle theme with the moon/sun icon in the top right corner. The setting is saved automatically.

☁️
Cloud Sync

Sign in with Google to sync your songs to the cloud.

02 — Editing

Writing Songs

Creating and Managing Songs

1

Open the side menu (☰) and click New Song to create a blank song.

2

Fill in the Title and Artist / Songwriter fields at the top of the side menu.

3

Edit the lyrics and chords in the main editor. Save with the Save button in the side menu.

4

Rename a song by changing the title field and clicking away — a confirmation dialog will appear.

CHORD Mode vs TEXT Mode

ProChorder has two editing modes, toggled with the CHORD button in the top bar:

ModeWhat you can do
CHORDClick existing chords to edit them with the radial chord builder. Drag and drop chords to reposition them.
TEXTFree text editing — type chords directly as [Am], [G], etc. inline with the lyrics.

Chord Builder (Radial)

Click a chord in CHORD mode to open the radial builder. Choose:

→

Root note — A, A#, B, C, C#, D, D#, E, F, F#, G, G#

→

Chord type — Major, m, 6, m6, 7, m7, 9, maj7, sus4, dim, aug and more

→

Bass note — optional alternate bass note (slash chords, e.g. G/B)

Sections

Add structural labels to a song via Sections in the side menu. Available sections: Intro, Verse, Chorus, Stick, Bridge, Solo, Outro, Dig.

Smart Links

Type a URL and press Space or Enter — it is automatically converted to a clickable link. You can also write [[url|text]] for custom link text.

Smart Paste

Paste text with chord lines above lyric lines (standard ChordPro format) — ProChorder detects and automatically merges them into inline chords.

Tip: Swipe left or right in the editor on touch devices to navigate to the next or previous song.

Edit History

ProChorder stores up to 100 history steps. Undo with Ctrl+Z / Cmd+Z and redo with Ctrl+Y.

03 — Live

Live Mode

Live mode is designed for performances — lyrics scroll automatically and the screen is kept awake (Wake Lock).

1

Press the LIVE! button in the side menu, or the floating ▶ button in the bottom right.

2

A control panel appears at the bottom: ▶ starts/pauses scrolling.

3

Adjust the scroll speed with the speed slider, or enter the song duration in minutes:seconds to calculate the speed automatically.

4

Navigate to the previous/next song with the ◀ and ▶ arrows in the control panel.

5

Exit Live mode with the ✕ Exit button.

Tip: Wake Lock keeps the screen active during Live mode — useful on mobile during performances. It activates automatically and resumes if the app returns to the foreground.
04 — Metronome

Metronome

The built-in metronome helps you keep time during rehearsals.

1

Open the side menu and click the metronome icon (the pendulum icon next to the LIVE! button) to start/stop.

2

Enter BPM directly in the number field, or use the + and − buttons next to it.

3

Valid BPM range: 40–300.

Tip: The font size selector (Size) and BPM control share the same row in the side menu for quick access.
05 — Tools

Built-in Tools

Tuner

Open via Tuner in the side menu. The tuner uses the device microphone to identify notes in real time. It stops automatically when the modal is closed.

Transpose

Open via Transpose in the side menu. Press +1 or -1 to shift all chords in the song up or down by one semitone. ProChorder automatically transposes all chords including slash chords.

Chord Mode

Toggles how chords are displayed — the button is in the side menu and mirrors the same function as the CHORD button in the top bar.

Font Size

Choose a size in the Size dropdown: 14, 16, 18, 20, 22, 24, 28 or 32 px. Useful for adjusting the display during performances.

06 — Setlist

Setlist Sharing

ProChorder makes it easy to share a setlist with band members via a short code.

Create a Setlist

1

Open Share setlist in the side menu.

2

Drag and drop songs from the Library column to the Setlist column (or click them) to select which songs to include.

3

Click Create code — a unique code is generated (e.g. ROCK99).

4

Share the code with your band.

Fetch a Setlist

1

Open Get Setlist in the side menu.

2

Enter the received code in the field and click Download.

3

The setlist loads and activates "Gig mode" — the project selector shows only the setlist songs.

Tip: In Gig mode the Exit Gig button (red) appears in the top bar. Click it to return to your full song library.

08 — Account & Cloud

Google Account & Cloud Sync

ProChorder works fully offline with local storage. Signing in with Google activates cloud synchronisation via Firebase.

1

Open the side menu and click Sign in with Google.

2

Authenticate in the popup window that appears.

3

Your songs are downloaded from the cloud and kept in sync.

4

Sign out with the Sign out button in the side menu.

Note: All saved songs are stored in the browser's localStorage regardless of login status. Signing in adds an extra cloud layer — your data is not lost if you sign out.
