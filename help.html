1. Manage your projects

Everything is saved locally in your browser. You don't have to click "Save" to switch songs, but it's a good idea to do so when you're done.

  • New song: Click on; + in the top menu. NOTE!! You must name your song in order to save.

  • Load song: Select a song from the drop-down menu in the top menu.

  • Save song: Open the side menu ‚ò∞ > Project > Save.

  • Rename song: Load the song, change the text in the "Song title" field in the side menu and click anywhere in the editor and press "Ok" in the dialog box that appears.

  • Remove song: Open ‚ò∞ > Project > Remove.

  • Remove ALL songs: Open ‚ò∞ > Project > Erase all.

 

 

2. Editing: Lyrics & Chords

This is the most important function of the app. You change mode with the  C / L-button in the top.

    C - Chord Mode:

    This is the default mode. Here the editor is "smart" and helps you with chords.

Create chords: By clicking on a word in the text, the chord selector opens.



Create Chords Manually inLMode

Type  [G7]Space. The chord is created automatically. Save your song when you're done. The next time you open the project the chord is in the right formate

  • Edit Chord: Click on the blue handle on an existing chord to change it. (Hover over the chord to make the handle appear)

  • Move Chord (Computer): Click and drag the small blue handle on a chord to move it.

  • Drag-Copy Chord: Hold down [alt], click and drag the small handle to the desired position.

  • L - Lyrics Mode:

    In this mode, the editor works like a regular text editor. You cannot click on words to create chords. This mode is perfect when you just want to type in long pieces of text without accidentally creating chords.

 

 

3. Section formatting (Verse, Chorus, etc.)

To structure your song:

  1. Place the text cursor on the line you want the verse, chorus, stick cursor to end up above "Solen skiner idag" (In english: "The sun is shining today":))..

  2. Open ‚ò∞ > Sections.

  3. Select "Verse" from the drop-down menu and click Insert.

  4. The line will now show "Verse" in larger text size and the text will move to the line below.

4. Links

Du can create clickable links in three different ways. Type the text below and press  Space.

  • Automatic link:

    • Type: www.ultimate-guitar.com

    • Result: www.ultimate-guitar.com

    • Short text link:

    Type: [[www.spotify.com/abba|Lissen]]

    Result: Lissen

     

    5. Tools

    Tools in the side menu ‚ò∞ > Edit.

    • Transpose +1 / -1: Raises or lowers all chords in the song by a half step.

    • Font size: Drag the slider to change the text size in the window.

     

     

    6. LIVE mode (Autoscroll)


    Perfect for playing live without having to touch the screen. NOTE! It is recommended that you exit live mode when you are going to change projects, work with lyrics or chords. Live mode is only for scrolling in a Live context!!

    1. Start LIVE-mode by using the side menue ‚ò∞ 

    2. Controls appears at the bottom.

    3. Play/Pause ‚ñ∂ / ‚ùö‚ùö: Starts and pauses scrolling. Spacebar also works.

    4. Speed Slider:Drag the slider to set the speed.

    5. Time m / s:You can also set an exact time e.g. 3 minutes 30 seconds that the song should take. The speed will then be calculated automatically.

    6. Exit:Exits LIVE mode. The Escape key also works..

     

     

    7. Import & Export

    You can easily backup and share your songs.

    • Export:

      • PDF: Creates a nice PDF of a song

      • TXT: Saves the song as a plain text file.

      • Song->JSON: Saves the current song in a special format that the app can read again. Use this for backup!

      • All->JSON: Saves all your songs in a single file. Perfect for total backup.

      • ZIP: Saves all your l&tars as separate PDF files in a ZIP file.

    • Import:

      • Imp.->JSON Select a .json file (that you previously exported) to load one or more songs.

      • Import URL Downloads a collection of songs directly from GitHub (used to load a collection of songs)..
